## 📌 Overview
This project provides an interactive analysis of global sales data, focusing on product performance, regional trends, and sales distribution.

## 🎯 Objectives
- Analyze total sales and order volume
- Identify top-performing products
- Compare sales across regions
- Understand category performance

## 📊 Key Insights
- Mobile Phones generated the highest sales
- South America recorded the top regional sales
- HP Laptop is the best-selling product
- Tablets category has the lowest performance

## 🛠 Tools Used
- Power BI
- Excel
